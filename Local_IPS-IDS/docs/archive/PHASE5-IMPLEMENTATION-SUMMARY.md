```
Cyber AI/Quantum
    ├── RabbitMQ (事件訂閱/發布)
    ├── PostgreSQL (數據儲存)
    ├── Redis (快取)
    ├── Prometheus (指標導出)
    └── Axiom UI (API 調用)

Axiom UI
    ├── Prometheus (監控數據)
    ├── Grafana (儀表板嵌入)
    └── Cyber AI/Quantum (AI 功能調用)

RabbitMQ
    ├── Pandora Agent (事件發布)
    ├── Axiom Engine (事件訂閱)
    └── Cyber AI/Quantum (AI 事件)
```

### 數據流

```
網路流量 → Pandora Agent → RabbitMQ → Cyber AI/Quantum
                                ↓
                          ML 威脅檢測
                                ↓
                         量子加密處理
                                ↓
                          AI 治理審核
                                ↓
                    RabbitMQ (檢測結果) → Axiom Engine
                                ↓
                         PostgreSQL (儲存)
```
